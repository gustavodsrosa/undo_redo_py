# undo_redo_py
Criação de um sistema simples de Desfazer / Refazer com o conhecimento de pilhas em Python, usando como base curso da Udemy

Atributos do sistema:
  - Permitir que o usuário adicione uma ou mais tarefas na pilha pré-definida
  - Permitir que o usuário tenha o retorno dessa pilha com as tarefas adicionadas por ele
  - Permitir que o usuário desfaça ou refaça algum comando de adicionar
